# Pharaphrasing

## break

```
Hello <br> World!
```
Hello <br> World!

## Indentation

```
- Hello
  - World
    - !!!
```

- Hello
  - World
    - !!!

# Text

## *Italics*

```
*Italics*
```
*Italics*

```
***Italics and Bold***
```
***Italics and Bold***

## <u>Underline</u>

```
<u>Underline</u>
```
<u>Underline</u>

## <span style = "color: blue">colors</span>

```
<span style = "color: red">Red</span>
```

# links

## inline links

```
[Google](https://www.google.com)
```
[Google](https://www.google.com)

## image links

```
![image]('/assets/image/ds_lab.png')](https://www.google.com)
```
![image]('/assets/image/ds_lab.png')](https://www.google.com)
